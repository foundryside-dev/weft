**Input** — the standard single-line text field (search box, id entry, filter value). Mono, overlay fill, strong border; focus paints the accent border plus a 1px accent ring. Pass `label` for an uppercase eyebrow above it.

```jsx
<Input label="Search" placeholder="Search issues…  ( / )" />
<Input label="Issue" defaultValue="PY-WL-101" width={180} />
```
