**Select** — a styled native `<select>` that matches the Input field, with a unicode ▾ caret. Use for short option sets (priority range, status filter).

```jsx
<Select label="Priority" defaultValue="P0–P1">
  <option>P0–P1</option>
  <option>P0–P4</option>
  <option>Ready only</option>
</Select>
```
