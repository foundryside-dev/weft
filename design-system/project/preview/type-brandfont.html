Install — bundled local, offline-safe
// fonts/SpaceGrotesk-Variable.ttf + fonts/SpaceGrotesk-OFL.txt
@font-face {
  font-family: 'Space Grotesk';
  src: url('fonts/SpaceGrotesk-Variable.ttf') format('truetype');
  font-weight: 300 700; font-display: swap;
}
 
// token — colors_and_type.css
--font-display: 'Space Grotesk', 'JetBrains Mono', ui-sans-serif, system-ui, sans-serif;
Do
  • Use it for the wordmark, hub hero, and slide titles.
  • Track display tight: -0.02em at 700.
  • Let the variable file cover 300–700 — no static cuts.
Don't
  • Set body, UI, code, or data in it — that stays mono.
  • Run it below the H1 role or in dense chrome.
  • Swap or add a display face — one brand voice only.
